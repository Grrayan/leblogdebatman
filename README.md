# Projet Le Blog de Batman

### Cloner le projet

```
git clone https://github.com/Grrayan/leblogdebatman.git
```

### Déplacer le terminal dans le dossier cloné
```
cd leblogdebatman_2027
```

### Installer les vendors (pour recréer le dossier vendor)
```
composer install
```

### Lancer le serveur
```
symfony serve
```
